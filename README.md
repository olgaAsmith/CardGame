# CardGame

## Project Description

This project is a card game built with modern web technologies, aiming to provide an engaging and interactive experience.

## Technologies Used

- React
- Next.js
- Tailwind CSS
- Zustand (for state management)
- TypeScript

## Installation and Setup

To set up and run the project on your local machine, follow these steps:

### 1. Clone the repository

```bash
git clone git@github.com:olgaAsmith/CardGame.git
cd CardGame
```

### 2. Install dependencies

```bash
npm install
# or
yarn install
```

### 3. Run the project

```bash
npm run dev
# or
yarn dev
```

The project will be accessible at `http://localhost:3000`.

## Project Structure

- `app/`: Main application logic, components, pages.
- `public/`: Static files (images, icons).
- `styles/`: Global styles.
- `components/`: Reusable React components.
- `store/`: Global application state using Zustand.
